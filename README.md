# G6

Relational data visualization framework.

[中文 README](README-zh_CN.md)

## Features

* Easy to use graph middleware
* Contains basic diagram editor

## Supporting environment

* Modern browsers and Internet explorer 9 and above

## Install

```bash
npm install @antv/g6
```
